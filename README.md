# FitMajster – aplikacja do planowania treningów i diety

FitMajster to aplikacja webowa wspierająca użytkowników w planowaniu treningów, zarządzaniu planami ćwiczeń oraz monitorowaniu postępów. Projekt powstaje jako część pracy inżynierskiej dotyczącej projektowania i implementacji nowoczesnych aplikacji internetowych wspierających aktywność fizyczną.

## Opis projektu

Aplikacja umożliwia:
- organizację i planowanie treningów,
- monitorowanie postępów użytkownika,
- zarządzanie planami ćwiczeń,
- budowanie nawyków aktywności fizycznej.

Projekt rozwijany jest iteracyjnie, a repozytorium przedstawia aktualny stan implementacji (wersja prototypowa / MVP).

## Kontekst pracy inżynierskiej

FitMajster powstaje w ramach pracy inżynierskiej z zakresu:
- projektowania aplikacji webowych,
- inżynierii oprogramowania,
- systemów wspierających zdrowy styl życia.

Cele pracy:
- zaprojektowanie architektury aplikacji,
- implementacja systemu zarządzania treningami,
- integracja backendu z bazą danych,
- stworzenie interfejsu użytkownika,
- przygotowanie fundamentów pod dalszy rozwój.

## Status projektu

Projekt znajduje się na etapie aktywnych prac rozwojowych.

### Zaimplementowane elementy
- struktura projektu (frontend + backend),
- podstawowa architektura aplikacji,
- konfiguracja środowiska developerskiego,
- pierwsze widoki interfejsu użytkownika,
- fundamenty logiki biznesowej,
- organizacja repozytorium i kodu.

Aktualna wersja stanowi bazę do dalszej rozbudowy.

## Planowane funkcjonalności

W kolejnych etapach przewidziano:
- system kont użytkowników (rejestracja / logowanie),
- tworzenie i edycję planów treningowych,
- bazę ćwiczeń,
- historię treningów,
- statystyki i wizualizacje postępów,
- pełną responsywność interfejsu,
- testy aplikacji,
- wdrożenie produkcyjne.

## Architektura (high-level)

FitMajster to aplikacja typu Full‑Stack Web Application, składająca się z:
- frontend: interfejs użytkownika (React + TypeScript),
- backend: logika biznesowa i API (Node.js + Express),
- baza danych: przechowywanie danych użytkownika (MySQL).

Architektura została zaprojektowana z myślą o modularności, skalowalności i łatwej rozbudowie.
